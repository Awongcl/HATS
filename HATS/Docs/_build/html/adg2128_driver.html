adg2128_driver module¶

adg2128_driver.I2C_ADDR_001 = 113¶

Protocol (24 bit ): I2C address -> data|Ax3-Ax0|AY2-AY0 - > xxxxxxLDSW data 0 = off, 1 = on x = dont care If LDSW = 1, the switch position changes after the new word is read. If LDSW = 0, the input data is latched, but the switch position is not changed. Setting LDSW on the last word allows all of the switches in that sequence to be simultaneously updated.

set_multiple_switch(control, list=[])¶

Set multiple switch state at once

Control options: “on”/”off”

list input : pairs of switches [x,y]

e.g [1,2,5,6,7,8] = X1&Y2 , X5&y6, X7&Y8

Register will latched during the sending sequence

Register will be update at once after the last command

Meaning all switches will be update simultaneously

Parameters

  • control (str) – “on” or “off”

  • list (list) – Pairs of switches, defaults to []
