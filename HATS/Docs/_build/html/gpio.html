gpio.dcdc_en(arr, bool=False)¶

Enable or disable on baord DCDC converters

Input a list of voltages, options: 5, 3.3 and 1.8 e.g to enabl3 5V and 3.3V = [5,3.3], bool = Teue

Parameters

  • arr (list) – List of voltage values

  • bool (bool) – On= True, Off = False, defaults to False
