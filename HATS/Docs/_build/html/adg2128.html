adg2128 module¶

adg2128.set_switch(device, control, x=0, y=0)¶

Set state of a single switch

Device options: 1,2

Control options: “on”,”off”

X options: 0-11 Y options: 0-7 :param device: Device :type device: int :param control: State :type control: str :param x: X :type x: int :param y: Y :type y: int

adg2128.set_multiple_switch(device, control, list=[])¶

Set multiple switch state at once

Device options : 1,2 Control options: “on”/”off”

list input : pairs of switches [x,y]

e.g [1,2,5,6,7,8] = X1&Y2 , X5&y6, X7&Y8

Register will latched during the sending sequence

Register will be update at once after the last command

Meaning all switches will be update simultaneously

Parameters

  • device (int) – Device

  • control (str) – “on” or “off”

  • list (list) – Pairs of switches, defaults to []

adg2128.read_switch(device, line)¶

Read back to is read the current state of a line e.g X1

When X0 is entered, this function returns a byte to indicate which Y is connected.

e.g Param = X0, returns 0x03 = 0000 0011

X0 is connected with Y0 and Y1 Line options: X0-7 Device options: 1,2

Parameters

  • device (int) – Device

  • line (str) – Line

Returns

2 Bytes, first byte = dummy ,second byte = data

Return type

List of bytes
